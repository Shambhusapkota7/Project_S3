Flexbox vs Grid (Layout Note)

Flexbox is one-dimensional: it is best for arranging items in a single row or a single column (for example, navigation bars or simple lists of cards).

CSS Grid is two-dimensional: it is designed to control both rows and columns at the same time, which is perfect for more complex layouts like project galleries.

In this portfolio, I used CSS Grid to arrange the Projects section (see the .projects-grid in CSS), because it lets me create a responsive grid of cards that automatically adjusts across rows and columns on different screen sizes.
